# Kişisel Finans Yönetim Uygulaması

## Açıklama
Bu bir kişisel finans takip uygulamasıdır.
Kullanıcılar gelir ve giderlerini kaydedebilir, aylık raporlar ve kategori bazlı
grafikler üzerinden finansal durumlarını takip edebilir.

## Kullanılanlar
- Python 
- Streamlit
- SQLite
- Pandas

## Kurulum
python -m venv venv
venv\Scripts\activate
pip install -r requirements.txt
